# Release checklist

This checklist tracks the remaining high-value work before a wider public beta.

## Done

- signed release APK build is working
- public-facing README and release notes exist
- issue-reporting guidance exists
- XTF import/export is working for shell-level launcher state
- reset-to-defaults flow is working
- browser node integration is working
- the default launcher surface has been cleaned up substantially

## Still worth doing before a wider beta

- one more review of default font/icon/SFX choices
- more XTF edge-case testing
  - missing apps
  - invalid or missing file access
  - corrupt packages
  - newer package versions
- final GitHub prerelease packaging and notes polish

## Later

- broader TV-oriented shell work
- broader platform integration cleanup
- deeper media-library polish
- later importer/theme exploration if still wanted
